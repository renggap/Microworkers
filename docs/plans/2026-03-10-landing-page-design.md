# Micro Worker Indonesia - Landing Page Design

**Date:** 2026-03-10
**Status:** Approved

## Project Overview

- **Name:** Micro Worker Indonesia
- **Type:** Single Landing Page
- **Tech Stack:** Next.js 14 (App Router) + Tailwind CSS
- **Location:** `/home/microworker/`

## Design System

### Style Reference
- **Porsche Design System** - Clean, premium, timeless
- **Accent Color:** Electric Blue (#00D4FF) - Shiny, modern

### Typography
- Sans-serif, minimal
- Large headlines (4-5rem for hero)
- Clean body text (16-18px)

### Color Palette
```
Primary Accent:  #00D4FF (Electric Blue)
Background Dark: #0a0a0a (Hero, Join, Footer)
Background Light: #ffffff (Benefits, How It Works)
Background Alt:   #f5f5f5 (How It Works section)
Text Primary:     #171717
Text Secondary:   #6b7280
Text Light:       #ffffff (on dark backgrounds)
```

## Page Sections

### 1. Hero Section
- Full viewport height (100vh)
- Abstract 3D shapes background
- Headline: "Connect. Work. Grow."
- Subheadline: "Platform micro worker Indonesia yang mempertemukan employer dengan talenta terbaik."
- CTAs: [Mulai sebagai Worker] [Cari Worker Sekarang]
- Navigation: Logo + "Tentang" "Cara Kerja" "Join"

### 2. Manfaat/Benefits Section
- White background
- 2 columns: Employer | Worker
- Clean cards with subtle shadows

**Employer Benefits:**
- ⚡ Akses Talent Cepat - Jalankan campaign Anda dengan cepat - worker siap mengerjakan dalam hitungan jam
- 💰 Biaya Transparan - Tanpa hidden fee, pembayaran per task
- 📊 Quality Assurance - Rating & review sistem untuk kualitas kerja

**Worker Benefits:**
- 💼 Fleksibilitas Penuh - Kerja kapan saja, di mana saja
- 💵 Pembayaran Cepat - Withdraw langsung ke rekening Anda
- 🚀 Kesempatan Berkembang - Bangun reputasi, tingkatkan skill

### 3. Alur Kerja / How It Works
- Light gray background (#f5f5f5)
- 2 flows: Employer & Worker
- 4 steps each with timeline

**Employer Flow:**
1. Post Campaign - Buat task dengan deskripsi jelas
2. Pilih Worker - Review profil dan pilih kandidat
3. Track Progress - Pantau pekerjaan secara real-time
4. Bayar & Selesai - Release payment setelah approve

**Worker Flow:**
1. Browse Task - Temukan campaign yang cocok
2. Apply & Accept - Submit proposal, tunggu approval
3. Kerjakan Task - Selesaikan sesuai deadline
4. Dapat Pembayaran - Withdraw ke rekening Anda

### 4. Join Section (CTA)
- Dark/electric blue gradient background
- 2 large cards: Employer | Worker
- Bold CTAs with hover effects

**Employer Card:**
- "Butuh talent untuk campaign Anda?"
- ✓ Akses 1000+ worker
- ✓ Mulai dalam 5 menit

**Worker Card:**
- "Mencari pekerjaan fleksibel?"
- ✓ Kerja dari mana saja
- ✓ Pembayaran cepat

### 5. Footer
- Dark background (black)
- 3 columns: Platform | Perusahaan | Support
- Copyright: © 2026 Micro Worker Indonesia
- NO social icons

## Responsive Design
- Desktop: Full layout with 2 columns
- Tablet: Stacked sections
- Mobile: Single column, optimized touch targets
